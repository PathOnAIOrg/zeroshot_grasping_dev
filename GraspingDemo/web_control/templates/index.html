üéì SO-101 Robot Control Tutorial

Step 1: Connect to Robot

1. Ensure robot is connected via USB (/dev/ttyACM0)

2. Click "Connect" button

3. Wait for green status indicator

Step 2: Set Home Position

1. Move robot to desired home position

2. Click "Set Home" to save current position

3. Use "Go Home" anytime to return

Step 3: Control Speed

Movement Speed: Controls home position and manual movements

Playback Speed: Controls trajectory replay speed

‚Ä¢ Range: 10% (very slow) to 200% (double speed)

‚Ä¢ Lower speed = smoother, safer movement

‚Ä¢ Higher speed = faster operation

Step 4: Record Trajectory

1. Click "Start" (motors auto-disable)

2. Physically move robot to desired pose

3. Click "Keyframe" to save position

4. Repeat for all waypoints (min. 2)

5. Click "Stop & Save" to finish

Step 5: Replay Trajectory

1. Click "Refresh" to load saved files

2. Click on trajectory to select

3. Click "Play" to execute once
